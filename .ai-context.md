# AI Development Context

## Project Type
Kubernetes Controller/Operator (Go)

## Key Technologies
- **Language**: Go 1.23.0
- **Framework**: controller-runtime v0.20.2
- **Kubernetes**: 1.32+
- **Build Tool**: Make
- **Testing**: Ginkgo v2 + Gomega

## Architecture Pattern
Custom Resource Definition (CRD) Controller following Kubernetes Operator pattern

## Development Standards
- Follow controller-runtime patterns
- Use structured logging
- Implement proper finalizers
- Follow Kubebuilder scaffolding conventions
- Use RBAC annotations for permissions

## Code Organization
```
api/v1/           # API types and CRD definitions
internal/         # Internal implementation
  controller/     # Controller reconciliation logic
cmd/             # Entry points
config/          # Kubernetes manifests
test/            # Test suites
hack/            # Build and development scripts
```
